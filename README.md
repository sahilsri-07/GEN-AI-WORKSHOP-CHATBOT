# 🏨 SmartStay Hotel Chatbot

A simple and interactive hotel booking chatbot built using **Flask, HTML, CSS, and JavaScript**.
This project was developed as part of a **Gen AI Workshop**.

---

## 🚀 Features

* 💬 Chatbot for hotel-related queries
* 🏙️ Supports multiple cities (Delhi, Mumbai, Pune)
* 🛏️ Room details with pricing
* 📅 Booking assistance through chatbot
* 🔗 Integrated with booking form
* 🌐 Multi-page website (Home, About, Login)
* 🎨 Clean and minimal user interface

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS, JavaScript
* **Version Control:** Git & GitHub

## ▶️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/sahilsri-07/GEN-AI-WORKSHOP-CHATBOT.git
```

2. Navigate to the project folder:

```
cd GEN-AI-WORKSHOP-CHATBOT
```

3. Install Flask (if not installed):

```
pip install flask
```

4. Run the application:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000
```
